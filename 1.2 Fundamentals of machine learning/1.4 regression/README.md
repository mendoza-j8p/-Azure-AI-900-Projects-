# Predicción de Ventas de Helados Usando Regresión Lineal

## Descripción

Este proyecto implementa un modelo de regresión lineal para predecir las ventas de helados en función de la temperatura máxima de un día. Utiliza un conjunto de datos históricos que contiene registros de temperaturas diarias y las correspondientes ventas de helados. 

## Objetivo

El objetivo de este ejercicio es:

1. **Entrenar un modelo de regresión lineal** con datos históricos de temperaturas y ventas de helados.
2. **Evaluar la eficacia del modelo** mediante métricas como el Error Absoluto Medio (MAE), el Error Cuadrático Medio (MSE), la Raíz del Error Cuadrático Medio (RMSE) y el Coeficiente de Determinación (R²).
3. **Visualizar los resultados** para entender la relación entre la temperatura y las ventas, así como las predicciones realizadas por el modelo.

## Resultados Esperados

Al final de la ejecución del script, se esperan los siguientes resultados:

- Un modelo capaz de predecir las ventas de helados basado en la temperatura.
- Cálculos de las métricas de evaluación que indiquen la precisión del modelo.
- Gráficas que muestran la relación entre las temperaturas y las ventas, así como las predicciones realizadas por el modelo.
